DMs[5] Analytic-DPM

  • 填年前（12.20左右）的坑（把截图里的公式给敲完，顺便熟悉一下之前的内容）

  • 摆了一整个寒假，唯一战果是一些拥有版权的封面图，比如上图：拍摄于东京新国立美术馆。

DDPM对应了diffusion SDE的maximum likelihood SDE solver，并且最优方差由Analytic-DPM来解析地给出。 maximum likelihood SDE solver: 某种一阶近似，而一阶近似其实存在无数种

回顾 DDIM

(1)q(x1:N∣x0)=q(xN∣x0)∏n=2Nq(xn−1∣xn,x0)q(xN∣x0)=N(xN∣α¯Nx0,β¯NI)q(xn−1∣xn,x0)=N(xn−1∣μ~n(xn,x0),λn2I)μ~n(xn,x0)=α¯n−1x0+β¯n−1−λn2⋅xn−α¯nx0β¯n

从（1）中可以看出，DDIM 并没有显式定义前向过程，而是在保证 q(xn∣x0) 与 DDPM 中一样的情况下，定义了逆向过程 q(xn−1∣xn,x0) （事实上，这个值在 DDPM 的推导中也要用到，作为逆向过程的 baseline）。在这个框架下，DDIM 和 DDPM 分别是 λn 取 0 和 β¯n−1β¯nβn 的特例。

因为其中有条件项 x0，所以上面的都还只是前向过程，该过程的逆向过程（也就是只给定 xn 的生成过程）一般定义为马尔可夫过程：

(2)p(x0:N)=p(xN)∏n=1Np(xn−1∣xn),p(xn−1∣xn)=N(xn−1∣μn(xn),σn2I),

其中，令 μn(xn) 具有和 μ~n(xn,x0) 相同的形式，只需要通过 score function 首先预测出 x0。

而对于逆向过程的 σn，DDIM 取 λn，而 DDPM 取 βn,β~n。都是人为设计的。

均值方差的解析解

(3)min{μn,σn2}n=1NLvb⟺min{μn,σn2}n=1NDKL(q(x0:N)∥p(x0:N)).

作者证明了这个优化目标的解析解：

(4)μn∗(xn)=μ~n(xn,1αn(xn+β¯n∇xnlog⁡qn(xn))),(5)σn∗2=λn2+(βnαn−βn−1−λn2)2(1−β¯nEqn(xn)∥∇xnlog⁡qn(xn)∥2d).

均值和原来的一样（也就是沿用 q(xn−1∣xn,x0) 是对的），但方差却有着不同的形式。

怎么求

λn,βn,αn 这些其实都是定死的，∇xnlog⁡qn(xn) 可以通过模型训练结果 sn(xn) 来估计，所以需要解决的只有 Eqn(xn)。用一个 Monte Carlo 采样近似一下即可：

(6)Γn=1M∑m=1M‖sn(xn,m)‖22d,xn,m∼iidqn(xn)

写成表达式就长这样：

(7)σ^n2=λn2+(β¯nαn−β¯n−1−λn2)2(1−β¯nΓn)

实验证明，M 取 10、100 也能获得不错的效果。

上下界

(8)|σn∗2−σ^n2|=(βnαn−βn−1−λn2)2⏟Coefficientβ¯n|Γn−Eqn(xn)∥∇xnlog⁡qn(xn)∥2d|⏟Approximation error.

方差的估计值和最优值的 bias （？）如上。注意到 approximation error 其实是固定的，而较短的去躁路径会导致系数项很大。为此，作者推导出了最优方差理论解的上下界，来对估计值进行裁切（上下界可以较为准确地计算，这样在实际计算方差时，可以直接算上下界），详见原论文。

路径最优化

对于 DDIM 中 1=τ1<⋯<τK=N ，共 K 个时间步的跳步，其均值、方差的最优解解析式如下：

(9)μτk−1|τk∗(xτk)=μ~τk−1|τk(xτk,1ατk(xτk+β¯τk∇xτklog⁡q(xτk)))στk−1|τk∗2=λτk−1|τk2+(βτkατk−βτk−1−λτk−1|τk2)2(1−β¯τkEq(xτk)∥∇xτklog⁡q(xτk)∥2d).

相较于之前的算法，这里对于固定的 {τi} 可以写出解析解，那么我们就可以进一步求解最优的 {τi}：

(10)minτ1,…,τKDKL(q(x0,xτ1,…,xτK)∥p∗(x0,xτ1,…,xτK))=d2∑k=2KJ(τk−1,τk)+c.

其中，J(τk−1,τk)=log⁡(στk−1∣τk∗2/λτk−1∣τk2) ，注意以下几点：

  • c 是一个和路径选择无关的常量
  • 可以通过MC来估计σ∗，从而使得 J(τk−1∣τk) 可计算

因此，这个问题就是一个计算 1→N 的最短路径的 DP 问题。

DPM-Solver

基本是跟着这篇文章读论文的，这里就不复述了。Take away message：将原本的微分方程，在一系列近似的基础上写成确定解的形式，从而摆脱了步长对误差的影响。如图中蓝线所示，步长会放大 dxt/dt 的误差，而改用 DPM-Solver 之后，误差可以看作是确定解基础上的偏移，与步长无关。
